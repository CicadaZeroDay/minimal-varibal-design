# Blog Articles

## Формат хранения статей

Статьи блога хранятся в двух форматах:

### 1. Markdown файлы (рекомендуется) ✅

Создайте файл `название-статьи.md` в этой папке с frontmatter:

```markdown
---
title: "Название статьи"
description: "Краткое описание для SEO"
author: "NMC Team"
publishedAt: "25 January 2025"
category: "Case Studies"
tags: ["tag1", "tag2"]
featuredImage: "/blog/image.jpg"
---

## Заголовок

Текст статьи в Markdown формате...

- Списки
- Работают
- Обычно

**Жирный текст** и *курсив* тоже поддерживаются.
```

### 2. TypeScript объекты (legacy)

Старые статьи хранятся в `legacy-articles.ts` для обратной совместимости.

## Как добавить новую статью

1. Создайте файл `content/articles/название-статьи.md`
2. Заполните frontmatter (метаданные)
3. Напишите контент в Markdown
4. Сохраните и закоммитьте

Статья автоматически появится на сайте!

## Преимущества Markdown

- ✅ Легко писать и редактировать
- ✅ Версионирование в Git
- ✅ SEO-дружелюбно
- ✅ Поддержка форматирования
- ✅ Не требует знаний программирования
